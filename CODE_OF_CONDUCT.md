# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

Be kind. Be specific. Listen. Show up.

Harassment, demeaning conduct, and bad faith are not welcome here. Disagreement is welcome, expressed in the same voice as the rest of the project — direct, considered, in good faith.

## Enforcement

Reports go to the founder: **krasumashi@ahtohmoh.com**. All reports are reviewed and investigated. Identities of reporters are protected.

Consequences scale with the conduct: a private conversation for a first incident, a temporary ban for repeated or serious incidents, a permanent ban for behaviour that endangers others.
